# UK Energy Insight

An interactive report exploring the UK Energy Paradox: Why are electricity prices high if renewables are cheap?

## Deployment

This project is a static HTML site designed to be deployed on Vercel.

### Structure

- `index.html`: The main application file containing HTML, CSS (Tailwind via CDN), and JS (Chart.js logic).
- `vercel.json`: Configuration for Vercel deployment.

## Technologies

- HTML5
- Tailwind CSS (CDN)
- Chart.js (CDN)
- Vanilla JavaScript
